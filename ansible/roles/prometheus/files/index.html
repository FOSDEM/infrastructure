Prometheus Live Demo

This is a live demo of the Prometheus monitoring system.

Components included:

  • Prometheus - Main server
  • Node Exporter - Machine metrics
  • Alertmanager - Alert aggregation and routing
  • Push Gateway - Batch jobs push metrics here

This is a simple single-machine setup. Check out the links above and on the menu to the left.

For more information about Prometheus see the official project documentation.

The Conway's Life in Prometheus live demo is also hosted here.
